## 🌟 GIỚI THIỆU TỔNG QUAN

Đây là Single-Page Portfolio chính thức của **Trần Ngọc Đạo** (`PRO Media Online • Digital Marketing • Data Analytics`), được xây dựng dựa trên tiêu chuẩn kiến trúc **Fire & Lightning Wireframe Framework v3**.

Thay vì những giao diện trang web cá nhân nhàm chán, portfolio này được thiết kế theo phong cách **Maximalist Glassmorphism (Tráng gương cảm ứng)** kết hợp ánh sáng tia sét và ngọn lửa, mang lại trải nghiệm thị giác ấn tượng nhưng vẫn tuân thủ nghiêm ngặt **"Quy tắc Không Gian Để Thở"** giúp bố cục siêu thoáng, sang trọng và dễ tiếp cận số liệu.

## ⚡ TÍNH NĂNG & HIỆU ỨNG NỔI BẬT (THEMING LAYER V3)

1. **🔥 Breathing Radial Atmosphere & CSS Ember Particles:**
   * Không gian Hero tự động thở với 2 lớp hào quang (`magma orange` góc dưới và `electric cyan` góc trên).
   * Hệ thống 12 hạt tàn lửa (`fl-e1` → `fl-e12`) bay lơ lửng và mờ dần được viết 100% bằng CSS thuần, không dùng thư viện nặng.

2. **⚡ Fracture Line SVG (`FractureLineSVG.jsx`):**
   * Các vệt sét nứt nẻ zigzag độc đáo nằm giữa các phần của trang web.
   * Sử dụng `IntersectionObserver` & `stroke-dashoffset` tự động **"vẽ nứt tia sét"** theo tốc độ cuộn trang của người xem.

3. **💎 Thẻ Card Tráng Gương với Viền Lửa Xoay (`.fl-card`):**
   * Mọi khối thẻ đều có lớp kính mờ `glass-mirror` với ánh gương quét chéo khi rê chuột.
   * Viền chìm (`::after`) xoay chuyển động `conic-gradient` 360 độ từ Cam → Vàng → Cyan → Tím (`2.5s/vòng`) đem lại cảm giác tối tân.

4. **⚡ Dòng Điện Giật Số Liệu (`Stats Jolt Counter`):**
   * Các chỉ số đếm tự động (`CountUp`) sử dụng phông chữ lập trình `JetBrains Mono`. Khi con số đáp đích, toàn bộ khối sẽ rung giật chớp nhoáng như có dòng điện chạy qua.

5. **🎯 Tương Tác Trình Diễn Thực Tế:**
   * **Timeline Quản Trị & Slide Hành Trình:** Hiển thị trục zigzag nối 4 giai đoạn sự nghiệp kèm các mốc phát sáng.
   * **Mô phỏng Ngân sách ROAS Tức thì (`ROAS Simulator`):** Cho phép người xem kéo thanh trượt ngân sách ads để tính toán tự động Doanh thu kỳ vọng, số đơn hàng và lợi nhuận ước tính ngay trên web.
   * **Showcase Dashboards & UI Demo:** Tích hợp trực tiếp các dashboard báo cáo kinh doanh thực tế từ GitHub (@daotnworking-hub).

## 🛠 CÔNG NGHỆ SỬ DỤNG (TECH STACK)

* **Core Framework:** React 18 + Vite (Siêu nhanh, tối ưu HMR).
* **Styling & Theming:** Tailwind CSS + Vanilla CSS Variables (Namespaced `--fl-*` và `--neon-*`).
* **Icons & UI:** Lucide React Icons + Custom SVGs + CSS Houdini Properties (`@property`).
* **Typography:** Google Fonts (`Bebas Neue` cho Display Hero, `Cabinet Grotesk` & `Outfit` cho Heading, `Inter` cho Body, và `JetBrains Mono` cho Số liệu).

## 🚀 HƯỚNG DẪN CÀI ĐẶT & CHẠY TRỰC TIẾP (LOCAL DEVELOPMENT)

### Yêu cầu hệ thống:
* Node.js (phiên bản 18+ hoặc mới nhất).
* npm hoặc pnpm/yarn.

### Các bước khởi chạy:

```bash
# 1. Clone repository về máy
git clone https://github.com/daotnworking-hub/digital-growth-portfolio.git

# 2. Di chuyển vào thư mục dự án
cd digital-growth-portfolio

# 3. Cài đặt các gói phụ thuộc (dependencies)
npm install

# 4. Khởi chạy máy chủ phát triển (Dev Server)
npm run dev
```

Sau khi chạy lệnh `npm run dev`, hãy mở trình duyệt và truy cập vào địa chỉ: `http://localhost:5173/` để xem website sống động.
